24 /// \file example1.cpp
25 ///
26 /// This is the first example program for the Maths3D library to show some
27 /// practical example usage of the vector functions.
28 ///
29 /// It creates a very simple scene containing 3 spheres, and then ray traces this
30 /// only to the first ray hit with no recursion or lighting.
31 ///
32 /// It sends out rays for each pixel of the output image. It defines the ray as
33 /// a pair of 3d points, the first is the eye position which is behind the center of
34 /// the view plane at a position of 0, 0, -viewDistance. It then has a second point
35 /// which is where the ray is pointing to which is through the pixel which is at
36 /// x, y, 0 for x values from -width/2 to width/2 and similarly for y and the height.
37 ///
38 /// Then for each ray, it tests it against each sphere to find the closest point
39 /// the ray gets to the center of the sphere, and then compares that to the radius
40 /// of the sphere. If it is less than the radius, then the ray is passing through
41 /// the sphere and the color of that sphere is used to draw the pixel.
42 ///
43 /// This ignores finding the closest sphere which would require additional calculations.
44

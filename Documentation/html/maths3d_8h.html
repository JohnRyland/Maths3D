Typedef Documentation

◆ Scalar1f

using Scalar1f = float

Scalar1f type is used through-out the Maths3D library as an alias to float so as to make it easy to swap it out for the double type or a fixed-point class.

Multiplies m1 with m2 and returns the resulting matrix that combines these. This can be used to compound together different transformations. Note the order matters, as a translation followed by a rotation, for example, is not the same transformation as a rotation followed by a translation.

Creates a matrix that when multiplied by it will be able to apply a 3D orthographic projection transformation.

Parameters
leftis the distance from the centre of the projection to the left edge of the view.
rightis the distance from the centre of the projection to the right edge of the view.
bottomis the distance from the centre of the projection to the bottom edge of the view.
topthe distance from the centre of the projection to the top edge of the view.
nearis the distance to the near plane. Depth values are scaled according to this.
faris the distance to the far plane. Depth values are scaled according to this.

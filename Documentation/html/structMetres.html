Detailed Description

The SI unit of metres for measuring distances.

Note
This helper type ensures it is harder to mess up the units and makes the APIs self-documenting.
As with degrees and radians, so too can distances be confusing. And it might also be confusing if a parameter is an angle or a distance, Take as an example a function like void setElevation(float elevation) It might mean the distance above sea level, which may be in meters or feet, or it might be an angle of elevation, such as 20 degrees above the horizon.
